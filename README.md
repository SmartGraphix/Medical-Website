# Medical Website

A responsive medical service website featuring a clean user interface, dark/light mode toggle, and smooth animations.

## Key Features

- **Responsive Design:** Fully adaptable layout for mobile, tablet, and desktop devices.
- **Dark/Light Theme:** Toggle between dark and light modes for user preference.
- **Smooth Animations:** Integrated ScrollReveal.js for engaging scrolling animations.
- **Interactive Elements:** Swiper.js for touch-friendly pricing cards and mobile navigation menu.

## Technologies Used

- **HTML5:** Semantic markup structure.
- **CSS3:** Custom styling with CSS variables, Flexbox, and Grid.
- **JavaScript:** Interaction logic for menu, theme toggle, and external libraries.
- **Libraries:**
    - [Remix Icons](https://remixicon.com/)
    - [Swiper.js](https://swiperjs.com/)
    - [ScrollReveal.js](https://scrollrevealjs.org/)

## How to Use

1.  Download the project files.
2.  Open `index.html` in your preferred web browser.
3.  Use the navigation menu to jump to different sections (Home, Delivery, About Us, Prices, Contact).
4.  Toggle the theme using the moon/sun icon in the header.

## Project Structure

- `index.html`: Main HTML structure.
- `styles.css`: Main stylesheet containing all designs and responsive rules.
- `main.js`: Main JavaScript file for functionality.
- `Images/`: Directory containing project assets.
